Markdown Diagrams Options

By default the extension sends your diagrams to the free public cloud instance https://kroki.io but you can install Kroki on your own infrastructure. Once you've installed Kroki, make sure to update the server URL to point to your instance.

Note: Mermaid, BPMN, Bytefield, Nomnoml, Svgbob, WaveDrom will still be rendered in SVG.

Background activity
More precise behavior at the expense of a little overhead.
On dynamic content websites, parses the page changes after a certain amount of time, lowering CPU usage.
Provides time for other extensions to render the markdown.
(local files use the file:// protocol - or no protocol)
